  • Key Features:

    -   Provides real-time insights into data flow, improving transparency and decision-making for businesses.

  • Technology Stack:

AIMD – Emergency Prediction App

  • Overview:

    AIMD (Artificial Intelligence for Medical Diagnostics) is an ongoing project aimed at revolutionizing healthcare by predicting medical emergencies in real time.

  • What It Does:

    The app forecasts potential medical conditions based on patient data, offering timely alerts and insights to healthcare providers.

  • Technology Stack:

    - Frontend: React, for an interactive user interface.
    - Backend: Python, for handling AI models and logic.
    - Database: MongoDB, to securely store patient data.
    - API Integration: OpenAI API, to improve prediction accuracy using natural language processing.

  • Development Goals:

    - Enhance patient care through proactive alerts.
    - Ensure data security and compliance with healthcare standards.
    - Integrate advanced analytics for better visualization of trends.

  • Future Plans:

    Expand the app’s scope to integrate wearable health data and refine prediction models for more specific medical scenarios.

RoboSub Mini-Float Robot

  • Overview:

    A mini-float robot designed for underwater data analysis, currently under development as part of the RoboSub competition.

  • Purpose:

    The robot aims to collect and process data in aquatic environments, offering insights into water quality, marine life, and environmental changes.

  • Technology Stack:

    - Microcontroller: ESP32, for efficient motor and sensor control.
    - Hardware: Arduino, for prototyping and system integration.
    - Sensors: Equipped with a suite of sensors to measure parameters such as temperature, salinity, and turbidity.

  • Key Contributions:

    - Designing control algorithms to ensure precise motor movements.
    - Developing a data collection system that interfaces with external platforms for real-time analysis.
    - Troubleshooting hardware challenges to improve the reliability of underwater operations.

  • Future Plans:

    Incorporate machine learning to analyze collected data and predict environmental patterns. Explore the use of more advanced hardware for deeper water exploration.
